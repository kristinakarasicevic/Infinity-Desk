# InfinityDesk

Full-stack web application built with Angular and ASP.NET Core.

---

## Backend
- ASP.NET Core Web API
- Entity Framework Core
- JWT Authentication
- CRUD operations (Notes, Tasks, Documents)

---

## Frontend
- Angular
- HTTP Client
- Routing
- JWT token handling

---

## Features
- User authentication (Login/Register)
- Notes management (CRUD)
- Tasks management (CRUD)
- Document upload & download
- User-based data separation (UserId / JWT)

---

## Tech Stack
- ASP.NET Core
- Angular
- Entity Framework Core
- SQL Server
- JWT
